parametrize_polygon(polygon, selection_polygon, offset_name='offsetx', origin=None)[source]¶

Parametrize pieces of polygon based on another polygon.

Parameters

  • polygon (Polygon) -- Polygon to parametrize.

  • selection_polygon (Polygon) -- Polygon to use as filter.

  • offset_name (str, optional) -- Name of the variable to create. Defaults to "offsetx".

  • origin (list, optional) -- List of x and y origin. Default is [0, 0]. It impacts the vector computation and is needed to determine expansion direction. If None, it will be computed from the polygon's center.

Returns

Returns True when successful.
