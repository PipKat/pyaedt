update_rlc_from_bom(bom_file, delimiter=';', valuefield='Func des', comptype='Prod name', refdes='Pos / Place')[source]¶

Function to update the edb_core components values (RLCs) with values coming from BOM. Files is a text file delimited and header values needed inside the bom reader have to be explicitly set if different from the defaults valuefield has to contain the value of the Component (eg. 22pF) at beginning of the value followed by a space and then the rest of the value

Parameters

  • bom_file -- full path to the bom file

  • delimiter -- delimiter default value ;

  • valuefield -- Field Header containing the value of component (Default value = "Func des")

  • comptype -- Field Header containing the type of component.eg "Inductor" (Default value = "Prod name")

  • refdes -- Field Header containing the RefDes of Component. eg "C100" (Default value = "Pos / Place")

Returns

True if the file contains the header and it is correctly parsed. the function returns True even if no values are assigned

Return type

type
